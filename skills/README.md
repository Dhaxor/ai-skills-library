# Skills Library

This directory groups skills by provider and by shared use case.

## Layout
- `openai/` OpenAI-focused skills and workflows
- `claude/` Claude-focused skills and workflows
- `shared/` vendor-neutral skills that work across models

## Principles
- Keep skills focused and reusable
- Prefer concise instructions plus source links
- Separate planning, execution, and verification
- Include examples and anti-patterns when helpful
