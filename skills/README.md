# Skills Library

This directory contains reusable skills organized by provider and use case.

## Principles
- Keep every skill focused and reusable
- Prefer concise instructions over long prose
- Separate planning, execution, and verification
- Include examples and anti-patterns

## Recommended layout
- `openai/` OpenAI-oriented skills
- `claude/` Claude-oriented skills
- `shared/` vendor-neutral skills
