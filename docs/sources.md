# Source Inventory and Methodology

This repository is curated from public first-party skill libraries and large community indexes. The goal is to provide a discoverable, SEO-friendly hub for OpenAI, Claude, and cross-model skills.

## Primary sources
- https://github.com/openai/skills
- https://github.com/anthropics/skills
- https://github.com/github/awesome-copilot
- https://github.com/microsoft/skills
- https://github.com/alirezarezvani/claude-skills
- https://github.com/VoltAgent/awesome-agent-skills
- https://github.com/ComposioHQ/awesome-claude-skills

## Method
- Prefer first-party repositories when available
- Use public repository trees and contents for stable indexing
- Deduplicate slugs across overlapping collections
- Keep provider-specific and shared skills separate
- Group skills by use case so users can search by intent

## Notes
- Counts in this repository reflect a snapshot of the sources above
- Community indexes may include duplicate or overlapping entries
- This repo aims to be a curated library, not a mirror of full upstream repositories
